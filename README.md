# Walmart Executive Sales Analysis

**Tool used:** Excel / Google Sheets

## Project overview
This project presents an executive sales analysis for Walmart using transactional and store-level data. The objective was to evaluate business performance through key commercial indicators and build a dashboard that supports decision-making.

## Business objective
The analysis focuses on identifying sales patterns, comparing department performance, and highlighting business insights that can help improve inventory planning, budget allocation, and commercial strategy.

## Dataset
The workbook includes the following data sources:
- `raw_ventas`
- `raw_departamento`
- `raw_tiendas`

It also includes working sheets for:
- `Resumen`
- `Dashboard`

## Key metrics analyzed
The project focuses on three main KPIs:
- **Efficiency**
- **Participation**
- **Volatility**

These indicators help evaluate department performance and sales behavior over time.

## What was done
- Reviewed and organized raw sales data
- Analyzed sales performance by department
- Calculated key business KPIs
- Built an executive summary
- Developed a dashboard in Excel for visual analysis
- Generated business insights and recommendations

## Main skills demonstrated
- Data cleaning
- Sales analysis
- Business KPI interpretation
- Executive reporting
- Dashboard design
- Data storytelling

## File included
- `Walmart_Executive_Sales_Analysis.xlsx`

## Project outcome
This analysis provides a business-oriented view of Walmart sales performance and supports decision-making with clear metrics, visual summaries, and executive insights.
